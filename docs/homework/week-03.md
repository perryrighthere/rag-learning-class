# Week 03 作业

这周只做一件事：

- 在老师给的 `pdf_parser` 服务基础上，补你自己的 PDF 后处理逻辑。

## 必做

1. 选一条路径跑通：
   - `MinerU API`
   - `pypdf`
   - `pymupdf`

2. 不改老师服务的职责边界。
   老师代码只负责：
   - 调 `MinerU API`
   - 或本地解析 PDF
   - 返回原始 JSON 或页文本

3. 你自己补下面至少一项：
   - 从 `MinerU` 返回结果里抽你需要的字段
   - 清洗页文本
   - 把结果整理成后面可送给 LLM 的输入结构

4. 写一个小说明，回答两件事：
   - 你走的是哪条路径
   - 你补的是哪一层，而不是哪一整套系统

## 不要做什么

- 不要把 Week 3 扩成完整 RAG。
- 不要提前做 chunking、retrieval、generation。
- 不要重写老师的 `pdf_parser` 服务边界。

## 提交物

- 你的改动代码
- 一份运行命令
- 一份输出样例
- 一段简短说明
